Cunha_Pope_AUS.ipynb was made using Google Colab. It is the code that contains the five techniques used to analyze the dataset.

weatherAUS.csv is the dataset. It was obtained from Kaggle via this link: https://www.kaggle.com/datasets/jsphyg/weather-dataset-rattle-package?datasetId=6012

If you are going to run this for yourself, place the weatherAUS.csv file in your Google Drive. Do not place it in any folders. 
Run the Cunha_Pope_AUS.ipynb file in Google Colab and allow it to access your drive.
